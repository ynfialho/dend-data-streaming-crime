## Pre requirements
* Python 3.7+
* Apache Spark 3.0.1
* Apache Kafka(standalone)

## Run

**1- Producer Python**
```shell
python3 kafka_server.py
```

**2- Consumer Python**
```shell
python3 consumer_server.py
```

**2- Consumer Spark Streaming**
```shell
spark-submit --packages org.apache.spark:spark-sql-kafka-0-10_2.12:3.0.1 --master local[*] data_stream.py
```
